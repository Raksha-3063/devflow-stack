# DevFlow 🚀

A modern Stack Overflow-inspired developer community platform built for asking questions, sharing knowledge, and helping developers grow together.

## ✨ Features

- 🔐 User Authentication
- 👤 Developer Profiles
- ❓ Ask & Browse Questions
- 💬 Answer System
- 🏆 Reputation Tracking
- 📊 Dashboard & Activity
- 🌙 Modern Dark UI
- ⚡ Real-time Backend Integration
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- React / Next.js
- Tailwind CSS
- TypeScript

### Backend & Database
- Firebase
- Supabase

### Other Tools
- Git & GitHub
- Antigravity
- Vercel (if deployed)
